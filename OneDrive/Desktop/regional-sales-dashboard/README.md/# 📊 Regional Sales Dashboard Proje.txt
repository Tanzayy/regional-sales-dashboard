# 📊 Regional Sales Dashboard Project

This project analyzes and visualizes regional sales data to identify daily trends and regional performance. It covers data cleaning, summary reports, and visual insights using Excel and Python.

---

## 🗂️ Project Structure

| Folder       | Description                                      |
|--------------|--------------------------------------------------|
| `data/`      | Cleaned CSV files and summary tables             |
| `assets/`    | Final charts and plots (PNG)                     |
| `notebook/`  | Jupyter Notebook used for data cleaning & EDA    |
| `docs/`      | Project documentation and summary writeups       |
| `dashboards/`| Excel dashboards with visual insights            |
| `BigQuery/`  | [If used] for SQL-based queries and outputs      |

## ✅ How to Use

1. Clone the repo  
   ```
   git clone https://github.com/Tanzayy/regional-sales-dashboard.git
   ```
2. Open the Jupyter notebook in `notebook/` to see the cleaning process  
3. Explore visualizations in the `assets/` folder  
4. Review final dashboard in the `dashboards/` folder
